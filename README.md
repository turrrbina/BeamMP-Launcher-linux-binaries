# BeamMP-Launcher-linux-binaries

I built these in incus containers, they *should* work - but no guarantee.

debian should cover debian, ubuntu, mint and other debian based distros.
fedora should cover bazzite.

- choose your distro
- download raw file
- create a directory like `/home/you/BeamMP-Launcher`
- put the file there, `chmod +x BeamMP-Launcher`
- enjoy
